How to contribute to code

  1. Navigate to directory parkWhere
  2. git pull to update code
git pull
  1. Checkout branch with appropriate name
  • Code maintenance, refactoring: dev/... e.g. dev/prettier
  • Adding feature: feat/... e.g. feat/googleLogin
  • Fix a bug: bugfix/... e.g. bugfix/wrongDateTime
git checkout -b dev/prettier
  1. Code!
  2. Stage your files to commit
git add .
# to add all files
  1. Commit files with a meaningful comment
git commit -m "made code pretty"
  1. Push code
git push
# You'll be prompted with
# The current branch dev/prettier has no upstream branch.
# Copy and paste the git push --set-upstream ...
git push --set-upstream origin dev/prettier
  1. Checkout back to the main branch
git checkout main
  1. Go to github and raise a pull request. Your work here is done
  2. Wait for code maintaner to approve your pull request and merge your branch with the main branch. Your branch will then be deleted. Don't forget to update your main branch to see the newly merged changes!
git pull
